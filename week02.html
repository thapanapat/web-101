All of web and web app are built using three fundamental Langauge technologies: HTML CSS and JavaSCript. These are the Langauge of the web.

In this post lets's focus on HTML. We will learn what HTML is about. and why do you should learn it.

What is HTML

HTML stands for Hyper Text Markup Langauge. It's a markup Langauge that web dev use to structure and descrube the content of the web page (Not a programming Langauge)

HTML consists of elements thats describe different type of content: paragraph, link ,headings, images , videos, etc. Web Broser understand HTML and render HTML code as a web.
